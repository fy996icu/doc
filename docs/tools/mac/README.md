# MAC使用技巧

## 快捷键

### 显示与隐藏隐藏文件

`Command + Shift + .`

### 聚焦搜索

`Command + 空格键`

### 全屏应用

`Control + Command + F`

### 切换应用

`Command + Tab`

### 同应用切换窗口

`Command + 反引号`

### 创建新文件夹

`Shift + Command + N`

### 隐藏或显示“访达”窗口中的路径栏

`Option + Command + P`

### 前往上一文件夹

`Command + [`

### 前往下一个文件夹

`Command + ]`

### 打开包含当前文件夹的文件夹

向上查看文件夹

`Command + 上箭头`
