# 日期处理

## 解析验证操作日期和时间

```sh
$ yarn add moment
```

**插件名称：** [Moment.js](https://www.npmjs.com/package/moment)

**在线示例：** [点击查看示例](https://www.momentjs.com.cn/)

![img](https://gitee.com/fy0829/pics/raw/master/example/momentjs.png)

$ yarn add dayjs
